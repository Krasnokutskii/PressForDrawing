# PressForDrowing
Show new drowing by tapping on the button 
## Tools implemented in project
- UIGraphicsImageRenderer
- .image()
- .pngData()
- .jpegData()
- setFillColor()
- setLineWidth()
- drawPath()
- context.cgContext
- UIBezierPath
- .translate(by:)
- .rotate()
